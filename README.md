# Checkout Flow

This project is a responsive multi-step checkout page built as part of a frontend assignment.

The goal was to create a clear and user-friendly checkout experience that collects personal information, shipping details, and payment information while providing validation feedback and handling payment states.

## Features

* Multi-step checkout flow
* Personal information form
* Shipping address form
* Payment details form
* Order review step
* Inline validation messages
* Loading state during payment submission
* Success and failure payment states
* Responsive design for mobile, tablet, and desktop

## Built With

* HTML5
* CSS3
* Bootstrap 5
* Vanilla JavaScript

## Running the Project

Simply clone the repository and open `index.html` in your browser.

No installation or additional dependencies are required.

## Design Decisions

I chose a step-by-step checkout flow to keep the form easy to follow and avoid overwhelming the user with too many fields at once.

Bootstrap was used for layout, responsiveness, and form components, while custom CSS was added to create a cleaner checkout experience and reduce the amount of styling code needed.

The checkout includes a review step before payment submission and a final confirmation screen to better reflect a real-world checkout process.

## Notes

This is a frontend-only project.

The payment process is simulated and no real payment information is processed or stored.
